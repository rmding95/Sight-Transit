APHont Installation Instructions: 

## macOS

1. Double-click each of four font files: Bold, Bold Italic, Italic, and Regular. 
2. Choose 'Install Font'. 

## Microsoft Windows

1. Find the 'Fonts' directory on your hard drive. For example, C:/WINNT/Fonts/
2. If you already have an older version of APHont in this folder, delete it before installing this version. 
3. Copy four font files to your computer's font directory: Bold, Bold Italic, Italic, and Regular. 

APHont is now ready to use!
